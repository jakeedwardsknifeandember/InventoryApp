\# Knife and Ember Inventory Management App



Welcome to the InventoryApp repository. This is a desktop-based software application built with Python to help track, manage, and audit inventory items, stock levels, and store resources.



\## Features

\* \*\*Inventory Tracking:\*\* Keep a real-time log of active ingredients, equipment, and supplies.

\* \*\*Database Management:\*\* Built-in scripts to test, verify, and safely reset application data structures.

\* \*\*Automated Safety Snapshots:\*\* Integrated local backup systems to prevent operational data loss.



\## File Structure

\* `main\_app.py` - The main program launch file.

\* `modules/` - Core folder containing the application's visual layout and logic rules.

\* `config.json` - System settings and environment configuration variables.

\* `requirements.txt` - The list of software packages needed to run this application.



\## How to Run the App

1\. Make sure you have Python installed on your machine.

2\. Install the necessary system dependencies:
